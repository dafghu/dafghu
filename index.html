Для чего нужен этот сайт?

Этот проект назначен на облегчение и ускорение процесса настройки и программирования дронов pioneer в самом начале пути. Здесь вы найдете:

  • Гайд на прошивку и настройку дрона для программирования
  • Всё нужное программное обеспечение
  • Небольшую документацию для библиотеки pioneer-sdk, которая поможет разобраться с основными атрибутами.

Кому он может помочь?

Этот сайт может помочь новичкам, которые только начинают свой путь в программировании дронов.
